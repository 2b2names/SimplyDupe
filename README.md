SimplyDupe is a lightweight Minecraft Paper plugin that adds one simple command: `/dupe`.

When a player runs `/dupe`, the plugin duplicates the item currently in their main hand one time. If the player’s inventory is full, the duplicated item is dropped naturally at their location.

Features:

* Simple `/dupe` command
* Duplicates the held item once
* OP-only permission by default
* Handles full inventories safely
* Adds sound and particle feedback after duplication

Permission:
`simplydupe.dupe`
